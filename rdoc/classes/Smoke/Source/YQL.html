Class Smoke::Source::YQL

Parent: Smoke::Origin

YQL will call to Yahoo YQL services

Usage: 

Smoke.yql(:ruby) do
  select  :all
  from    "search.web"
  where   :query, "ruby"
end

where (column, value)
